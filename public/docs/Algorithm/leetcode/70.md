# 70. 爬楼梯

## 题目

假设你正在爬楼梯。需要 n  阶你才能到达楼顶。

每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？

注意：给定 `n` 是一个正整数。

```
示例 1：

输入： 2
输出： 2
解释： 有两种方法可以爬到楼顶。
1.  1 阶 + 1 阶
2.  2 阶

示例 2：

输入： 3
输出： 3
解释： 有三种方法可以爬到楼顶。
1.  1 阶 + 1 阶 + 1 阶
2.  1 阶 + 2 阶
3.  2 阶 + 1 阶
```

## 题解

### 动态规划

`f(n)` 是爬 `n` 阶台阶的方法，可以得到状态转移方程：

> f(n) = f(n-1) + f(n-2)

边界条件：

- `f(1) = 1`
- `f(2) = 2`

```JavaScript
/**
 * @param {number} n
 * @return {number}
 */
var climbStairs = function (n) {
  if (n <= 2) return n;
  let a = 1,
    b = 2,
    ans = 0;
  while (n-- > 2) {
    ans = a + b;
    a = b;
    b = ans;
  }
  return ans;
};

```
